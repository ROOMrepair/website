本节介绍如何管理集群可见性。集群可见性用于控制集群授权给哪些企业空间。

* 您可以将一个集群授权给多个企业空间，从而允许企业空间中的资源运行在该集群上。

* 您也可以将多个集群授权给同一个企业空间，从而在企业空间中创建联邦项目。有关联邦项目的更多信息，请参阅link:../../../11-use-extensions/04-kubefed[联邦项目]。
