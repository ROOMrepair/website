* 通用参数
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|规则名称
|告警规则的名称。规则名称可包含任意字符，最长 63 个字符。

|标签设置
|为告警规则设置键值对标签，以便在通知订阅中通过标签过滤，发送标签符合特定条件的告警通知。

告警规则创建后，会自动生成一个键为 rule_id 的标签。

|概要
|告警消息的内容概要。消息概要可包含任意字符，最长 63 个字符。

|详情
|告警消息的详细描述。消息详情可包含任意字符，最长 256 个字符。
|===
