* 对于**蓝绿部署**，在新版本或旧版本右侧点击**接管**可将业务流量全部转发给该版本。

* 对于**金丝雀发布**任务，您可以选择指定新旧版本的流量分配比例，或根据请求参数将请求转发给新版本或旧版本。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|指定流量分配
|拖动滑块可设置新旧版本接收业务流量的百分比。

|指定请求参数
|将参数满足特定条件的请求转发给新版本，其他请求转发给旧版本。
|===
--

流量镜像任务将业务流量的副本发送给新版本进行测试，而不实际暴露新版本，所以不需要设置业务流量转发策略。
